## Tech Stack

- **Vite** + **TypeScript** + **React**
- **shadcn-ui** + **Tailwind CSS**
- **React Router** for navigation

## Quick Start

```bash
# Clone and install
git clone https://github.com/kapillamba4/kapillamba4.github.io.git
cd kapillamba4.github.io
npm install

# Development
npm run dev

# Build
npm run build
```
